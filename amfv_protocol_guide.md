## Protocol: RTNM (Record→Task→Navigate→Map)

### Step 1: RECORD Protocol
- **Before entering location:** `record on`
- **At location:** Perform required actions
- **After actions:** `record off`
- **Verification:** Confirm recording captured

### Step 2: TASK Analysis
- Identify task type at location
- Determine required actions
- Note any puzzle elements
- Mark task completion criteria

### Step 3: NAVIGATE Efficiently
- Use known navigation routes
- Follow compass direction protocols
- Track position mentally
- Avoid backtracking when possible

### Step 4: MAP Maintenance
- Update mental map with new locations
- Note connections and routes
- Mark completed recording locations
- Track remaining target locations
