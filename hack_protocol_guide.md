# Hack Protocol Guide: Threat Assessment and Survival Framework

## Universal Framework Transfer: From Wumpus to Hack
Based on systematic thinking and threat assessment principles validated in Hunt the Wumpus, this guide adapts protocols for Hack's survival challenges.

## Core Principles Adapted for Hack

### 1. Systematic Threat Assessment → Monster/Environment Scanning
**Every Turn:** Assess immediate surroundings
- Visible monsters and their distance
- Available escape routes (corridors, doors)
- Environmental hazards (traps, pits)
- Resource proximity (items, food, potions)

### 2. Special Move/Ability Check → Action Priority Matrix
**Decision Hierarchy:** Always choose safest available action
1. Wait/heal (s) when HP low and area safe
2. Retreat (move away from threats)
3. Attack only when advantage clear
4. Use items only when identified/necessary

### 3. Catastrophe Prevention → HP Conservation Protocol
**Critical Thresholds:**
- HP < 30%: Immediate retreat/heal priority
- HP < 50%: Avoid combat unless overwhelming advantage
- HP < 75%: Conservative movement only
- HP = 100%: Can take calculated risks

### 4. Resource Vulnerability Scan → Inventory Management
**Key Rules:**
- Never drop unidentified potentially useful items
- Always maintain food reserve if possible
- Potions: Test only when safe (full HP, isolated)
- Weapons: Keep best available weapon wielded

### 5. Positional Evaluation → Map Awareness
**Essential Awareness:**
- Current level and approximate position
- Known safe areas and danger zones
- Route back to safer areas
- Stair locations (up/down)

## Quick Decision Loop for Hack

### STOP
- Freeze after each turn output
- Don't rush next command

### SCAN
1. HP status and immediate threats
2. Monster proximity and types
3. Available movement directions
4. Visible items/resources

### EVALUATE
1. Is immediate combat necessary?
2. Can I safely wait/heal?
3. What's the safest retreat path?
4. Should I use any items now?

### SIMULATE
1. Mental simulation of each possible move
2. Worst-case outcome for each option
3. Best survival probability choice
4. Resource conservation considerations

### COMMIT
1. Execute chosen command
2. Monitor results carefully
3. Update mental map
4. Repeat loop

## Protocol: TASH (Threat→Assess→Survive→Heal)

### Step 1: THREAT IDENTIFICATION
- List all visible/immediate threats
- Rate threat level (1-5, 5 = lethal)
- Note monster attack patterns observed
- Mark threat directions on mental map

### Step 2: ASSESS OPTIONS
- Generate all possible actions
- Rate each for survival probability
- Consider resource usage
- Evaluate position improvement

### Step 3: SURVIVE (Execute Safest)
- Choose highest survival probability action
- Execute with confidence
- Monitor results immediately
- Be ready to abort/adjust

### Step 4: HEAL (Recovery Priority)
- After threat neutralized, seek healing
- Use s turns in safe areas
- Consume healing items if necessary
- Return to ≥80% HP before further exploration

## HP Management Protocol
**Observed Pattern:** Passive healing occurs but is slow

### Healing Strategy:
1. Find confirmed safe area (no recent monsters)
2. Use consecutive s turns
3. Monitor HP increments (5→6→7 observed)
4. Continue until HP ≥ 80% or threat appears
5. If healing stalls at specific HP, consider moving slightly

### Emergency Protocol (HP < 30%):
1. Immediate retreat to known safe zone
2. Use all available escape options
3. If trapped, use emergency items (scrolls, potions)
4. Last resort: Fight desperately

## Inventory Protocol for Tourist Class

### Priority Items:
1. Food rations (survival essential)
2. Healing potions (emergency use)
3. Offensive items (darts, wands)
4. Defensive items (armor if found)
5. Utility items (keys, tools)

### Identification Protocol:
1. Test only when HP full and isolated
2. One item at a time
3. Note effects carefully
4. Update mental inventory database

## Weight/Gate Management Protocol
**Problem:** "You are carrying too much to get through"

### Solution Protocol:
1. Immediately drop heaviest non-essential items
2. Priority drop order: camera → food rations → pancakes
3. Mark drop location on mental map
4. Retrieve later if possible/safe

### Optimal Carry Set:
- Weapon in hand
- 1-2 emergency potions
- Minimal food (1-2 rations)
- Essential quest items only
